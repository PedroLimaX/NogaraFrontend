Pagina Web simple para SOCEP

Esta pagina web fue realizada en el año 2019 como proyecto integrador hecho con el fin de dufundir la informacion importante para la cooperativa SOCEP la cual buscaba mostrar informacion sobre sus productos y servicios, ademas de sus eventos e informacion destacada de la cooperativa, era un proyecto planeado para realizarse en 4 meses pero se hizo en 2 semanas xd

Lenguajes utilizados: HTML + CSS + PHP + SQL

Sitio Web para Gestion de Proyectos escolares

Este sitio web fue diseñado en el año 2021 para gestionar los proyectos de los alumnos del campus San Jose Chiapa del CRC de la Benemerita Universidad Autonoma de Puebla, el cual tenia como objetivo principal tener un expediente de cada proyecto realizado en colaboracion con empresas externas en donde se mantiene el registro de los proyectos con sus fases, cada fase con sus entregables y cada entregable con sus requerimientos, se debia gestionar los alumnos, docentes y empresas involucrados ademas de apartados para registrar los recursos disponibles para los proyectos y para guardar los archivos evidencia de cada proyecto, el proyecto se debia realizar en 5 meses

Lenguajes utilizados: HTML + CSS + PHP + SQL

Tienda en linea para SG Iluminacion

La tienda online hecha para SG Iluminacion en 2021 debia contener el catalogo de los productos de cada proveedor registrado, el sitio debia permitir gestionar los proveedores, darle seguimiento a los pedidos, notificar por correo electronico a los proveedores y a los clientes del estado de su pedido, el administrador podia gestionar los usuarios y contenido general del sitio, los proveedores podian editar el perfil de su empresa y ademas, actualizar su catalogo de productos mediante un archivo csv y tambien podian generar un reporte de ventas mensual. Este proyecto debia realizarse en 4 meses.

Lenguajes utilizados: HTML + CSS + PHP + Javascript + SQL
